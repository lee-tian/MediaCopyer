## 下载

- **macOS**: [MediaCopyer-v1.1.1.dmg](https://github.com/lee-tian/MediaCopyer/releases/download/v1.1.1/MediaCopyer-v1.1.1.dmg)

## 安装说明

### macOS
1. 下载 `MediaCopyer-v1.1.1.dmg`
2. 双击打开DMG文件
3. 将 `MediaCopyer.app` 拖拽到 Applications 文件夹
4. 在 Applications 文件夹中找到并运行 MediaCopyer

## 系统要求

- macOS 10.13 或更高版本
- Python 3.6+ (仅源码安装需要)
