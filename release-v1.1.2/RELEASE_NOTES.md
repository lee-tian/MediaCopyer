# MediaCopyer v1.1.2

发布日期: 2025-11-01

## 更新内容

- 修复自动发布卡住问题
- 改进发布脚本稳定性
- 添加无交互自动发布模式
- 增加超时处理和错误报告
- 提供多种发布脚本选择

## 下载

- **macOS**: [MediaCopyer-v1.1.2.dmg](https://github.com/lee-tian/MediaCopyer/releases/download/v1.1.2/MediaCopyer-v1.1.2.dmg)

## 安装说明

### macOS
1. 下载 `MediaCopyer-v1.1.2.dmg`
2. 双击打开DMG文件
3. 将 `MediaCopyer.app` 拖拽到 Applications 文件夹
4. 在 Applications 文件夹中找到并运行 MediaCopyer

## 系统要求

- macOS 10.13 或更高版本
- Python 3.6+ (仅源码安装需要)
